## 1. Goal

Ship the first set of working, interactive calculators wired to the pure finance
engine (`src/lib/finance`) and Redux Toolkit. These are the foundational tools that
the larger Resale (Story 1) and New Launch (Story 2) projections will reuse.

## 3. Finance Engine Additions (from this + prior step)

Already implemented and unit-tested (14 tests passing):

- `Buyer`, `PurchaseInputs`, `DownPaymentBreakdown`, `ValidationResult` types.
- `absdRate`, `jointAbsdRate`, `calcABSDForBuyers` — joint ABSD = highest applicable
  rate among co-owners (SG rule).
- `breakdownDownPayment(inputs)` — pools cash/CPF, computes down payment, loan, BSD,
  joint ABSD, total upfront, and funding allocation (cash covers the 5% minimum first).
- `validatePurchase(inputs)` — UI-friendly issues: `DP_BELOW_MIN`, `CASH_BELOW_MIN`,
  `INSUFFICIENT_FUNDS`, `BUYER_COUNT`, `STAMP_DUTY_FUNDS` (warning).
- `assertValidPurchase(inputs)` — throwing variant.

## 4. Architecture

### 4.1 New files
```
src/
  store/slices/purchaseSlice.ts     # down payment inputs (1-2 buyers), persisted
  components/ui/
    Card.tsx                        # section container
    Field.tsx                       # label + control wrapper
    NumberInput.tsx                 # numeric input with prefix/suffix
    Money.tsx                       # SGD formatter
    index.ts                        # barrel
  app/
    tools/
      down-payment/page.tsx         # joint buyer down payment calculator
      stamp-duty/page.tsx           # BSD + ABSD
      mortgage/page.tsx             # instalment + amortization
```

### 4.2 State strategy
- **Down Payment**: backed by RTK `purchase` slice (persisted) — it's the richest
  form and benefits from persistence across sessions.
- **Stamp Duty / Mortgage**: local component state (lightweight, transient) computed
  directly from the pure engine functions. No need to persist.

### 4.3 Data flow
```
UI input → purchaseSlice (or local state) → lib/finance pure fns → live breakdown/validation
                   ↓
            redux-persist → localStorage
```

---

## 5. Singapore Rules Enforced
- Minimum **25%** down payment (LTV 75%).
- Minimum **5% cash** of price (CPF cannot satisfy this).
- **Joint ABSD** uses the highest applicable rate among all co-owners.
- BSD tiered bands; ABSD by profile × nth purchase.
